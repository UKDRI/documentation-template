## Using Mermaid.js

If your Markdown tool supports **Mermaid.js** (like GitHub Pages, MkDocs, or Docusaurus), you can create dynamic charts like pie charts, flowcharts, and more.

### Pie Chart:
```mermaid
pie
    title Task Distribution
    "Completed": 50
    "In Progress": 30
    "Pending": 20
```
---
### Flowchart:
```mermaid
graph TD
    A[Start] --> B[Do Work]
    B --> C{Decision?}
    C -->|Yes| D[Continue]
    C -->|No| E[Stop]
```
